## 「予想屋がいない場」はあるか（重要）
- **本メモ/JSONで“いない”と断定した場はゼロ。** unknown は「今回未確認」であって不在の確定ではない。
- 予想屋は高齢化・後継者不足で**減少傾向**（開場から70年級のベテランも普通の世界）だが、まだ多くの場に残る。減った結果いない場も理屈上はあり得るが、**名指しで“予想屋ゼロ”と確定できる資料は確認できなかった**。
- 補強材料: 予想屋LIVE配信『SODA』の配信対象＝**戸田・多摩川・三国・蒲郡**（＋びわこの可能性）。オンライン化で在籍が可視化された場もある。
