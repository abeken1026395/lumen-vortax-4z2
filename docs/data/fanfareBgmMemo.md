# 場内BGM・ファンファーレ 調査ログ（fanfareBgm.json 用）

- 作成日: 2026-07-07
- 対象: 24場の締切BGM・場内テーマ曲・イメージソング／全国共通ファンファーレ
- 方針: **検証可能な事実のみ**。曲名・作曲者名の記載はOK、歌詞・音源の転載はNG。
  出典URL＋確認日をセットで記録。**公式サイトで曲名を確認できた場のみ** `fanfareBgm.json` の `venues` に収録し、
  二次ソース（ファンブログ・メディア等）止まりの場は本メモに「未確認」として残すのみ（JSONには入れない）。

---

## ⚠️ 実行環境の制約（重要・要ローカル再確認）

本調査を行った Claude Code 実行環境は egress ポリシーが厳しく、**ボートレース公式ドメインへ直接アクセスできない**。

- `www.boatrace.jp` / `www.boatrace-suminoe.jp` など **`boatrace.jp` 系・`boatrace-*.jp` 系はすべて CONNECT 403**（プロキシがポリシー拒否）で本文取得不可。
- 二次メディア `macour.jp`（`sp.macour.jp`）、キャッシュの `web.archive.org` も同様に遮断。
- 唯一 **Web検索（検索エンジンの要約）だけが通る**状態。よって「公式ページ本文を自分の目で確認して曲名を写す」作業が今回はできていない。

このため今回の `fanfareBgm.json` は：

- `common.fanfare` … 複数ソースで裏取りできる周知の事実として収録（曲名なし・作曲者3名）。
- `venues` … **タスク指示（プロジェクトオーナー）が公式ページと曲名を明示していた住之江のみ**を収録。

**次アクション（オーナー / ネットワーク制約のないローカル環境向け）:**
下の「二次ソースで曲名が挙がっている場」の各行を、対応する**公式サイトの音源/メロディ/BGMページで実際に確認**し、
確認できた場だけ `fanfareBgm.json` の `venues` に `title / type / source(公式URL) / checked` を付けて追記する。
